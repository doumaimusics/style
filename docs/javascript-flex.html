Flex 兼容

当浏览器不支持 CSS 的 flex 属性时，会在相应元素上的 class 属性中增加名为flex的样式名。并根据设定的子元素数量再增加一个名为flex_n（n 为数字）的样式名。

该功能仅仅只是增加了一个样式名，具体兼容还需要在 CSS 文件中设置。

CSS 参考设置：

.footer_nav.flex_1 nav a{width:100%;}
.footer_nav.flex_2 nav a{width:50%;}
.footer_nav.flex_3 nav a{width:33%;}
.footer_nav.flex_4 nav a{width:25%;}
.footer_nav.flex_5 nav a{width:20%;}

默认#footer_nav、.nav_tab元素已设置兼容

如有其他元素需要兼容，可以通过变量window.GLOBAL.fixFlexSelectors设置：

window.GLOBAL.fixFlexSelectors = [
  {
    name: '#yourid', // 父元素选择符名称，如：'#id_name', '.class_name', ' element_name' 等
    tag: 'a' // 子元素选择符名称，根据子元素的数量，再增加 flex_n 的样式名
  },
  {
    name: '.yourclass',
    tag: 'li'
  }
]
